# Breast Cancer Classification

## Maincraft AI/ML Internship Task 4

### Objective

- Classification
- Confusion Matrix
- Precision
- Recall
- F1 Score
- ROC Curve
- Class Imbalance

### Dataset

Breast Cancer Dataset (Scikit-Learn)

### Models

- Logistic Regression
- Decision Tree Classifier

### Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Results

| Model | AUC Score |
|------|-----------|
| Logistic Regression | **0.9954** |
| Decision Tree | 0.9157 |

### Best Model

The Logistic Regression classifier achieved the highest ROC-AUC score (0.9954), indicating excellent discrimination between malignant and benign tumors. It was selected as the final model due to its superior performance and better generalization capability.
